# Problemas técnicos e instabilidade

Palavras-chave: erro, bug, não carrega, fora do ar, lento, travando, aplicativo

Se o sistema está apresentando erros ou lentidão:

1. Verifique nossa página de status em status.exemplo.com para saber se
   há instabilidade geral em andamento
2. Limpe o cache do navegador ou atualize o aplicativo para a versão mais
   recente
3. Teste em outra rede (ex.: 4G em vez do Wi-Fi) para descartar problema
   de conexão local

Erros persistentes com código (ex.: ERR-500, ERR-403) devem ser reportados
ao suporte técnico com um print da tela e o horário aproximado em que
ocorreram.
